# Tungsten Edge 钨极 · v0.1.2

## What changed

- **New app icon** — simplified to a single 钨 character; smaller sizes show just the letterform.
- **Divider between the messaging zone and other apps** — makes the two sections easier to tell apart at a glance.

## Requirements

- macOS 12.0 (Monterey) or later
- Intel and Apple Silicon (universal binary)

## Install

### Homebrew

```bash
brew tap moonbai-studio/tungsten-edge
brew trust moonbai-studio/tungsten-edge
brew install --cask tungsten-edge
```

### Direct download

1. Download `Tungsten-Edge-0.1.2.zip`, unzip, drag **Tungsten Edge** into Applications.
2. **First launch:** right-click (Control-click) → **Open** → Open again (not yet notarized).
3. Grant **System Settings → Privacy & Security → Accessibility**.

## Known limitations

- Not signed/notarized → first launch needs right-click → Open.
